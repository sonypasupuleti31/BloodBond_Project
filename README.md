# BloodBond – Smart Blood Bank Management System

BloodBond is a web-based blood bank management system developed to simplify and
digitize the process of managing blood donors, receivers, and blood inventory.

The system provides a centralized platform where donors can register, receivers
can request blood, and administrators can manage blood stock efficiently.

## Technologies Used

- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat Server

## Modules

### Admin Module
- Manage donor records
- Manage receiver requests
- Update blood stock
- Monitor system activities

### Donor Module
- Donor registration
- Update donor details
- View donation history

### Receiver Module
- Request blood
- Check blood availability
- Track request status

## Features

- Secure login authentication
- Real-time blood availability tracking
- Efficient donor and receiver management
- Centralized database storage
- Easy-to-use interface

## Purpose of the Project

The main objective of this project is to improve communication between donors,
receivers, and blood banks while maintaining accurate blood inventory records
and reducing manual management effort.

## Database

MySQL is used as the backend database to store donor, receiver, and blood stock data.

## Server Used

Apache Tomcat Server
